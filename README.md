# CineTrack
Movie tracking app built with React.

## Features
-Search movies (TMDB API)

-Track status:
  -Want to Watch
  -Watching
  -Watched

-Select favorite movies


## Tech Stack
-React

-JavaScript

-CSS

-TMDB API
